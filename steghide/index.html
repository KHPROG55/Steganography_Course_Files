أهم ما في أداة Steghide :
الملفات المدعومة :
JPG(JPEG), BMP, WAV, MP3, M4A.
embed :
تضمين(إخفاء ملف مخفي داحل ملف آخر)
extract :
إستخراج ملف مخفي من داخل ملف مخفي آخر
info :
لمعرفة معلومات عن cover أو stego file
encinfo :
لإظهار لسته من الخوارزميات المدعومة

قسم embed :
-cf :
Cover File
-ef :
Embed File(Hiddin File)
-e :
Encryption(لتضمين ملف داخل ملف بإستخدام خوارزمية تشفير معينة)
إذا تبي التضمين بدون تشفير اكتب :
-e none
-z :
حرف z سمول .. لضفط الداتا المخفية بعد التضمين
مستوى الضغط : من 1 إلى 5 .. 1 أسرع .. 9 أفضل
-Z :
حرف Z كابيتال .. إذا ما تبي ضغط الداتا اكتبه بدال z سمول
-p :
passphrase(password)

قسم extract :
-sf :
Stego File(تحديد ملف stego ﻷستخراج الداتا منه)
-xf :
Extract File(لتسمية الملف المخفي المستخرج)
-p :
passphrase(password)
